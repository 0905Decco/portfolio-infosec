# Derrick Kibiwot - Portfolio

A professional, single-page portfolio website built with React (No-Build).
Theme: Data Protection & Privacy Compliance (Navy Blue & Gold).

## 🚀 How to Run

### Option 1: Automatic (Recommended)
Double-click **`start.bat`**.
This will:
1.  Find Python on your computer.
2.  Start a local server.
3.  Open the website in your browser.

### Option 2: Manual
Open a terminal and run:
```bash
python server.py
```

### Option 3: Direct File Access
You can also just double-click **`index.html`** to view the site directly.

## 📁 Project Structure
- `index.html`: The complete application (Code + Styles).
- `server.py`: Python script to run a local server.
- `start.bat`: Launcher for Windows.
- `legacy_backup/`: Your previous portfolio files.

## 🛠 Tech Stack
- **React 18** (CDN)
- **Tailwind-like Custom CSS**
- **Lucide Icons**
